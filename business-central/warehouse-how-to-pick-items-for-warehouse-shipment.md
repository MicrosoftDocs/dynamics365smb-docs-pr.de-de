    <span data-ttu-id="8cc2f-119">Wenn Sie an einer bestimmten Kommissionierung arbeiten möchten, wählen Sie die Kommissionierung in der Liste aus, oder filtern Sie die Liste, um die Kommissionierungen zu finden, die speziell Ihnen zugeordnet wurden.</span><span class="sxs-lookup"><span data-stu-id="8cc2f-119">If you need to work on a particular pick, select the pick from the list or filter the list to find the picks that have been assigned to you specifically.</span></span> <span data-ttu-id="8cc2f-120">Öffnen Sie die Kommissionierungskarte.</span><span class="sxs-lookup"><span data-stu-id="8cc2f-120">Open the pick card.</span></span>

<span data-ttu-id="8cc2f-130">Zusätzlich zur Kommissionierung für Herkunftsbelege, die in diesem Thema beschrieben wird, können Sie Artikel ohne Bezug zu Herkunftsbelegen an Lagerorten entnehmen und einlagern.</span><span class="sxs-lookup"><span data-stu-id="8cc2f-130">In addition to picking for source documents, as described in this topic, you can take and place items between bins without referring to source documents.</span></span> <span data-ttu-id="8cc2f-131">Weitere Informationen finden Sie unter [Kommissionieren und Einlagern ohne Herkunftsbeleg](warehouse-how-to-create-put-aways-from-internal-put-aways.md).</span><span class="sxs-lookup"><span data-stu-id="8cc2f-131">For more information, see [Pick and Put Away Without a Source Document](warehouse-how-to-create-put-aways-from-internal-put-aways.md).</span></span>
