# <a name="receive-items"></a><span data-ttu-id="521b9-103">Empfangen von Artikeln</span><span class="sxs-lookup"><span data-stu-id="521b9-103">Receive Items</span></span>
<span data-ttu-id="521b9-104">Wenn Artikel bei einem Lager ankommen, das nicht für die Bearbeitung des Wareneingangs eingerichtet wurde, können Sie den Wareneingang einfach im zugehörigen Geschäftsdokument, wie einer Einkaufsbestellung, einer Verkaufsreklamation oder ein eingehender Umlagerungsauftrag, erfassen.</span><span class="sxs-lookup"><span data-stu-id="521b9-104">When items arrive at a warehouse that is not set up for warehouse receipt processing, you simply record the receipt on the related business document, such as a purchase order, a sales return order, or an inbound transfer order.</span></span>

<span data-ttu-id="521b9-105">Wenn Artikel bei einem Lager ankommen, das für die Bearbeitung des Wareneingangs eingerichtet wurde, müssen Sie die Zeilen des freigegebenen Herkunftsbelegs abrufen, der ihren Wareneingang ausgelöst hat.</span><span class="sxs-lookup"><span data-stu-id="521b9-105">When items arrive at a warehouse that is set up for warehouse receipt processing, you must retrieve the lines of the released source document that triggered their receipt.</span></span> <span data-ttu-id="521b9-106">Wenn Sie Lagerplätze nutzen, können Sie entweder den eingetragenen Standardlagerplatz akzeptieren oder, wenn der Artikel nie zuvor in diesem Lager verwendet wurde, tragen Sie den Lagerplatz ein, in den der Artikel eingelagert werden soll.</span><span class="sxs-lookup"><span data-stu-id="521b9-106">If you have bins, you can either accept the default bin that is filled in, or if the item has never been used before in the warehouse, fill in the bin where the item should be put away.</span></span> <span data-ttu-id="521b9-107">Sie müssen dann die Mengen der Artikel eingeben, die Sie erhalten haben, und den Wareneingang buchen.</span><span class="sxs-lookup"><span data-stu-id="521b9-107">You must then fill in the quantities of the items you have received, and post the receipt.</span></span>

2. <span data-ttu-id="521b9-112">Öffnen Sie eine bestehende Bestellung, oder erstellen Sie eine neue.</span><span class="sxs-lookup"><span data-stu-id="521b9-112">Open an existing purchase order, or create a new one.</span></span> <span data-ttu-id="521b9-113">Weitere Informationen finden Sie unter [Erfassen eines Einkaufs](purchasing-how-record-purchases.md).</span><span class="sxs-lookup"><span data-stu-id="521b9-113">For more information, see [Record Purchases](purchasing-how-record-purchases.md).</span></span>

    <span data-ttu-id="521b9-115">Der Wert im Feld **Bereits gelief. Menge** wird aktualisiert.</span><span class="sxs-lookup"><span data-stu-id="521b9-115">The value in the **Qty.     <span data-ttu-id="521b9-132">Alle Zeilen des freigegebenen Herkunftsbelegs, die die Filterkriterien erfüllen, werden nun auf der Seite **Wareneingang** eingefügt, in dem Sie die Filterfunktion aktiviert haben.</span><span class="sxs-lookup"><span data-stu-id="521b9-132">All released source document lines that fulfill the filter criteria are now inserted in **Warehouse Receipt** page from which you activated the filter function.</span></span>
