## <a name="to-move-components-to-an-operation-area-in-basic-warehouse-configurations"></a><span data-ttu-id="c87f7-123">So lagern Sie Komponenten in einen Arbeitsgangbereich in Basis-Lagerkonfigurationen um</span><span class="sxs-lookup"><span data-stu-id="c87f7-123">To move components to an operation area in basic warehouse configurations</span></span>  
1.  <span data-ttu-id="c87f7-124">Wählen Sie das Symbol ![Nach Seite oder Bericht suchen] (media/ui-search/search_small.png "Nach Seite oder Bericht suchen") aus und geben Sie **Lagerbestandsumlagerung** ein. Wählen Sie dann den relevanten Link aus.</span><span class="sxs-lookup"><span data-stu-id="c87f7-124">Choose the ![Search for Page or Report](media/ui-search/search_small.png "Search for Page or Report icon") icon, enter **Inventory Movement**, and choose the relevant link.</span></span>  
2.  <span data-ttu-id="c87f7-125">Füllen Sie auf dem Inforegister **Allgemein** das Feld **Nr.** aus.</span><span class="sxs-lookup"><span data-stu-id="c87f7-125">On the **General** FastTab, fill in the **No.**</span></span> <span data-ttu-id="c87f7-126">Feld</span><span class="sxs-lookup"><span data-stu-id="c87f7-126">field.</span></span> <span data-ttu-id="c87f7-127">Sie können die EINGABETASTE betätigen, um in den Nummernserien eine Auswahl zu treffen.</span><span class="sxs-lookup"><span data-stu-id="c87f7-127">You can press the Enter key  to select from the number series.</span></span>  
3.  <span data-ttu-id="c87f7-128">Geben Sie im Feld **Lagerortcode** den Lagerort ein, an dem die Umlagerung stattfindet.</span><span class="sxs-lookup"><span data-stu-id="c87f7-128">In the **Location Code** field, enter the location where the movement occurs.</span></span>  
4.  <span data-ttu-id="c87f7-129">Wählen Sie die **Herkunftsbelege holen** Aktion aus.</span><span class="sxs-lookup"><span data-stu-id="c87f7-129">Choose the **Get Source Documents** action.</span></span> <span data-ttu-id="c87f7-130">Füllen Sie alternativ das Feld **Herkunftsbeleg** aus, und klicken Sie anschließend auf die Schaltfläche **AssistEdit** im Feld **Herkunftsnr.**.</span><span class="sxs-lookup"><span data-stu-id="c87f7-130">Alternatively, fill in the **Source Document** field, and then choose the **AssistEdit** button in the **Source No.** field.</span></span>  
5.  <span data-ttu-id="c87f7-131">Wählen Sie im Fenster **Herkunftsbelege** den Montageauftrag aus, für den Sie Komponenten umlagern möchten, und klicken Sie anschließend auf die Schaltfläche **OK**.</span><span class="sxs-lookup"><span data-stu-id="c87f7-131">In the **Source Documents** window, select the assembly order that you want to move components for, and then choose the **OK** button.</span></span>  

    <span data-ttu-id="c87f7-132">Für alle erforderlichen Komponenten, die umgelagert werden können, werden im Fenster **Lagerbestandsumlagerungen** eine Entnahme- und eine Einlagerungszeile generiert.</span><span class="sxs-lookup"><span data-stu-id="c87f7-132">For each needed component that can be moved, one Take line and one Place line are generated in the **Inventory Movements** window.</span></span> <span data-ttu-id="c87f7-133">Alle Felder mit Ausnahme des Felds **Bewegungsmenge** werden entsprechend den Herkunftsbelegzeilen vorbelegt.</span><span class="sxs-lookup"><span data-stu-id="c87f7-133">All fields except the **Qty. to Handle** field are prefilled according to the source document lines.</span></span> <span data-ttu-id="c87f7-134">Das Feld **Bewegungsmenge** weist Null aus, bis Sie die Menge angeben, die Sie tatsächlich umgelagert haben.</span><span class="sxs-lookup"><span data-stu-id="c87f7-134">The **Qty. to Handle** field is set to zero until you enter the quantity that you have actually moved.</span></span>  

    <span data-ttu-id="c87f7-135">Sie können den Lagerplatzcode in einer Entnahmezeile aber nur entsprechend der Verfügbarkeit ändern.</span><span class="sxs-lookup"><span data-stu-id="c87f7-135">You can change the bin code on a Take line but only according to availability.</span></span> <span data-ttu-id="c87f7-136">Wenn Sie in einer Entnahmezeile die Schaltfläche **AssistEdit** im Feld **Lagerplatzcode** auswählen, wird das Fenster **Lagerplatzinhalt** geöffnet und zeigt nur Lagerplätze an, an denen die Komponente verfügbar ist.</span><span class="sxs-lookup"><span data-stu-id="c87f7-136">If you choose the **AssistEdit** button in the **Bin Code** field on a Take line, then the **Bin Contents** window opens and only shows bins where the component is available.</span></span>  

    <span data-ttu-id="c87f7-137">Sie können den Lagerplatzcode in einer Einlagerungszeile nicht ändern.</span><span class="sxs-lookup"><span data-stu-id="c87f7-137">You cannot change the bin code on a Place line.</span></span> <span data-ttu-id="c87f7-138">Nur der Lagerplatzcode, der in der Komponentenzeile des Herkunftsbelegs definiert ist, wird angenommen.</span><span class="sxs-lookup"><span data-stu-id="c87f7-138">Only the bin code that is defined on the component line of the source document is accepted.</span></span> <span data-ttu-id="c87f7-139">Dies unterstützt das Prinzip, dass die Rolle, die eine Komponente anfordert, hier ein Montagearbeiter, weiß, wo die Komponente platziert werden muss.</span><span class="sxs-lookup"><span data-stu-id="c87f7-139">This supports the principle that the role who requests a component, which is an assembly worker in this procedure, knows where the component must be placed.</span></span> <span data-ttu-id="c87f7-140">Wenn Sie die Komponenten an einem anderen Lagerplatz einlagern möchten, müssen Sie zuerst den Lagerplatzcode in der Komponentenzeile ändern und die Lagerbestandsumlagerungszeilen dann neu erstellen.</span><span class="sxs-lookup"><span data-stu-id="c87f7-140">If you want to place the components in a different bin, then you must first change the bin code on the component line and then re-create the inventory movement lines.</span></span>  
6.  <span data-ttu-id="c87f7-141">Geben Sie im Feld **Bewegungsmenge** die vollständige oder einen Teil der Menge ein, die Sie tatsächlich umgelagert haben.</span><span class="sxs-lookup"><span data-stu-id="c87f7-141">In the **Qty. to Handle** field, enter the full or partial quantity that you have actually moved.</span></span> <span data-ttu-id="c87f7-142">Die Werte in den Zeilen "Entnahme" und "Einlagerung" müssen gleich sein.</span><span class="sxs-lookup"><span data-stu-id="c87f7-142">The value on the Take and the Place lines must be the same.</span></span> <span data-ttu-id="c87f7-143">Andernfalls kann die Lagerplatzumlagerung nicht gebucht werden.</span><span class="sxs-lookup"><span data-stu-id="c87f7-143">Otherwise, you cannot register the movement.</span></span>  

    > [!NOTE]  
    >  <span data-ttu-id="c87f7-144">Wie bei anderen Lageraktivitäten, können Sie die Einlagerungszeile aufteilen, indem Sie die Aktion **Aktionen** und dann die Aktion **Zeile aufteilen** auswählen.</span><span class="sxs-lookup"><span data-stu-id="c87f7-144">As in other warehouse activities, you can split the Place line by selecting the **Actions** action and then choosing the **Split Line** action.</span></span> <span data-ttu-id="c87f7-145">In diesem Fall muss die Summe der beiden geteilten Einlagerungszeilen der Menge in der Zeile für die Entnahme entsprechen.</span><span class="sxs-lookup"><span data-stu-id="c87f7-145">In that case, the sum of the two split Place lines must equal the quantity on the Take line.</span></span>  

7.  <span data-ttu-id="c87f7-146">Wenn Sie bereit sind, die Umlagerungen zu erfassen, die Sie durchgeführt haben, wählen Sie die Aktion **Lagerbestandsumlagerung registrieren**.</span><span class="sxs-lookup"><span data-stu-id="c87f7-146">When you are ready to register the movements that you have performed, choose the **Register Invt. Movement** action.</span></span>  

    <span data-ttu-id="c87f7-147">Die Lagerplatzposten werden erstellt und zeigen an, dass die Komponenten jetzt an den Lagerplätzen vorhanden sind, die in den Montageauftragszeilen angegeben sind.</span><span class="sxs-lookup"><span data-stu-id="c87f7-147">Warehouse entries are created reflecting that the components now exist in the bins specified on the assembly order lines.</span></span>  

    > [!NOTE]  
    >  <span data-ttu-id="c87f7-148">Anders als beim Umlagern von Komponenten mit einer Lagerkommissionierung, wird Verbrauch nicht gebucht, wenn Sie eine Lagerbestandsumlagerung erfassen.</span><span class="sxs-lookup"><span data-stu-id="c87f7-148">Unlike when you move components with an inventory pick, consumption is not posted when you register an inventory movement.</span></span> <span data-ttu-id="c87f7-149">Dieser Schritt muss separat ausgeführt werden, indem Istmeldung und Verbrauch des Montageauftrags gebucht werden.</span><span class="sxs-lookup"><span data-stu-id="c87f7-149">That step must be performed separately by posting the assembly order output and consumption.</span></span> <span data-ttu-id="c87f7-150">Weitere Informationen finden Sie unter Montageauftrag.</span><span class="sxs-lookup"><span data-stu-id="c87f7-150">For more information, see Assembly Order.</span></span>
